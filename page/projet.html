My Projects

Over my years of programming and a mind always buzzing with ideas, I have completed many projects. The most notable ones are listed below:



Project: Anubis

This project was requested by a friend attending a competing school. Her team had almost abandoned her, so I helped from A to Z.
The goal was to create an ARG as a website (HTML, CSS, JavaScript) with an accompanying video.
The code is available on my GitHub (link in the Profile).
Duration: 2 weeks

Project: Pravda

Pravda is the name of my web server hosted on a Raspberry Pi 5.
I currently use it to host multiple completed projects and store data for associations and other purposes.
Duration: 1 week

Project: Ministorium

As the lead programmer of the Warhammer club (Warhagine), I was asked to code a website to archive played games and compile the rules of different game modes, with the ability to edit articles.
Duration: 3 weeks

Project: Division

One of the projects closest to my heart as it is linked to the biggest coding project of my life.
Division is a connected watch where SSH and other coding options would be available.
I created it because the only watches that natively support this are from Google and Apple, and as a student, I do not have the funds to buy them.
Duration: 2 months (not yet finished)
